# 키키아이 과제
---
## 실행 방법
1. npm i
2. npm start
---
## 프로젝트 구성
- React + Vite(번들러)
- tailwind(utility css library)
---
## 사용 라이브러리
1. svgr/rollup - svg 컴포넌트화
2. react-query - 페이지 최적화
3. react-beautiful-dnd - 드래그 앤 드롭
4. moment - 타임 유틸
5. recoil - 전역 상태 관리
6. react-transition-group - 페이지 전환 애니매이션
